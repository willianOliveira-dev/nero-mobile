# Nero Mobile — Suggested Commands

## Development
```bash
# Start Expo dev server
pnpm start

# Start on Android
pnpm android

# Start on iOS
pnpm ios

# Start web
pnpm web
```

## Code Quality
```bash
# Lint
pnpm lint
```

## API Generation
```bash
# Regenerate API client from OpenAPI spec (Orval)
pnpm generate-api
```

## System (Windows)
- `dir` instead of `ls`
- `findstr` instead of `grep`
- `type` instead of `cat`
- PowerShell is the default shell
